# Customer Churn Analysis

# Project Overview
Analyzed customer data to identify factors contributing to customer churn and provide insights to improve customer retention strategies.

# Tools & Technologies
- SQL (Data Analysis, Aggregations)
- Power BI / Excel
- Data Visualization

# Dataset
Customer dataset containing demographic details, tenure, contract type, payment method, and churn status.

# Key Analysis
- Churn rate calculation
- Churn distribution (Yes vs No)
- Churn by contract type, payment method, and tenure
- Customer behavior analysis

# Dashboard Features
- KPI Cards (Total Customers, Churn Rate)
- Churn Distribution Pie Chart
- Churn by Contract Type (Bar Chart)
- Churn by Payment Method (Bar Chart)
- Tenure vs Churn Analysis

# Key Insights
- Customers with month-to-month contracts had the highest churn
- Electronic check payment method showed higher churn rates
- Customers with lower tenure were more likely to churn
- Higher monthly charges contributed to increased churn risk

# Conclusion
The project highlights key churn drivers and helps businesses design targeted strategies to improve customer retention.
